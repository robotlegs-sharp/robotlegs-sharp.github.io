Overview

Robotlegs is an Application Framework. It offers: Dependency, Module, Command and View management.

Robotlegs Sharp

Robotlegs sharp is a port of the orignal Robotlegs for Actionscript 3.0. It's goal is to say as close to the orignal framework only to be expanded in order to fit the new language and platforms.

It should stay true to the original framework by being clean, programmer friendly, testable and expandable.
